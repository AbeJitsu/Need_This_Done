## Admin-Only Features

- View ALL projects (not just own)
- See client email on project cards
- Filter by client email
- Change project status
- Add internal notes (hidden from users)
- Delete projects

---

## Files Involved

- `app/components/AdminDashboard.tsx` - Main dashboard component
- `app/components/ProjectCard.tsx` - Individual project cards
- `app/components/ProjectDetailModal.tsx` - Project detail view with admin controls
- `app/api/projects/all/route.ts` - API endpoint for all projects

---

## Accessibility

- All interactive elements have focus states
- Color contrast meets 5:1 ratio
- Status badges use both color AND text
- Screen reader friendly structure
- Filter controls have proper labels
