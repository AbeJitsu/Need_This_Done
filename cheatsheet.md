### When to Use TDD

| Scenario            | Use TDD? | What kind of test?           |
|---------------------|----------|------------------------------|
| Fixing a bug        | YES      | Test that fails WITH the bug |
| New feature         | YES      | Test expected behavior       |

| New component       | YES      | Accessibility + behavior     |
| Config changes only | MAYBE    | If behavior changes          |
